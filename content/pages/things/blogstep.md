# BlogSTEP
Static site generator with web-based file manager and markdown editor. Interface inspired by [NeXTSTEP](https://en.wikipedia.org/wiki/NeXTSTEP) and [Window Maker](https://en.wikipedia.org/wiki/Window_Maker).
